
## Time Tracking Dashboard

This project is a time tracking dashboard that displays how much time is spent on different activities (like work, play, study, etc.) over daily, weekly, and monthly periods. It was built as a practice project to improve frontend development skills.

### What I Did
- Built the layout using HTML and CSS
- Practiced using CSS Grid and Flexbox for responsive design
- Used JavaScript to update the dashboard dynamically based on user selection
- Loaded data from a JSON file to populate the dashboard

### What I Learned
- How to structure a responsive layout with CSS Grid and Flexbox
- How to manipulate the DOM with JavaScript to update content
- How to work with external data (JSON) in a frontend project
- Improved my understanding of organizing files and assets for a real project
